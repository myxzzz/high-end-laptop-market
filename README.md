## 0. 项目背景
本项目旨在探索 AI PC 浪潮下，硬件成本波动（DDR5/PCIe 5.0）如何转化为整机市场的价值转嫁。我们构建了一套“竞争-价值-心智”三位一体的分析框架，尝试穿透硬件定价的迷雾。

## 1. 技术路线与实现思路
项目通过思睿智训平台构建了全链路分析流，涵盖从原始特征治理到深度模型预测的过程。

### A. 数据治理 (Data Governance)
- **去重与平滑处理**：对同一型号的不同配置进行逻辑去重，采用“评价数×售价”估算产值规模。
- **降噪逻辑**：主动剔除 GPU 变量（因其采用不透明的 B2B 定价），专注市场化程度最高的内存与硬盘。
- **标准化**：采用 **Z-Score 标准化** 处理量纲差异。

### B. 核心算法模型
- **K-Means 聚类**：基于欧氏距离将市场划分为 4 个价值簇。固定随机种子 (Seed 15) 以保证研究的可重复性。
- **决策树 (CART)**：以 **Gini 系数** 为分裂指标，提取高端市场的“准入准则”。发现 `is_AI_Hardware` 是区分溢价区间的全局根节点。
- **NLP 文本挖掘**：引入 **8 倍传播权重加权法**，从 23 万条评论中捕捉被频次稀释的“智慧交互”等弱信号。

### C. 理论创新：影子溢价模型 (Shadow Premium Model)
通过“差值溢价法”，在同一模具下对比不同配置的差值，计算核心组件的影子价格基准，从而剥离非硬件集成价值。

## 3. 已知局限性与缺陷
1. **数据深度**：未能深入穿透显卡芯片、主板模具等 B2B 定价环节的动态成本。
2. **时效性**：影子价格基准受全球供应链波动影响较大，结论具有时效性。
3. **样本偏差**：评论数据来自主流电商平台，可能存在幸存者偏差。
